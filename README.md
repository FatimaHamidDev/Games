# 🎮 Kids Number Learning Games (JavaScript)

An interactive collection of browser-based games designed to help young kids (age 4–7) learn numbers, spelling, and basic keyboard/motor skills in a fun and engaging way.

---

## 🌟 What Makes This Special?

These are not just games — they are **learning experiences**:

* 🔢 Learn numbers (1–10)
* 🔤 Practice spelling (ONE, TWO, THREE…)
* ⌨️ Improve keyboard familiarity
* 🧠 Boost memory and recognition
* 🎉 Instant feedback with sound & animations

---

## 🕹️ Games Included

### 1️⃣ Keyboard Number Explorer

* Kids press the **correct keyboard letters**
* Guided step-by-step spelling
* Voice assistance + hints
* Great for early typing skills

### 2️⃣ Number Spelling Adventure

* Drag-and-drop letters into correct positions
* Improves **hand-eye coordination**
* Reinforces spelling through interaction
* Fun animations and celebration effects 🎊
